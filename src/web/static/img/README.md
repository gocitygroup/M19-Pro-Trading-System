# Logo Files

This directory contains the logo files for the Gocity Trader Trading Profit Monitoring application.

## Required Files

1. **logo1.png** - Main Gocity Group logo (displayed in the header, recommended size: 200x50px)
2. **logo2.png** - Secondary logo (displayed in the top right, recommended size: 160x40px)

## Instructions

1. Replace the placeholder logo files with your actual Gocity Group logos
2. Ensure the files are named exactly as specified:
   - `logo1.png` for the main logo
   - `logo2.png` for the secondary logo
3. Recommended formats: PNG with transparent background
4. The application will automatically hide logos if files are missing

## File Specifications

- **Format**: PNG (preferred) or JPG
- **Background**: Transparent PNG recommended
- **Main Logo (logo1.png)**: 200x50px or similar aspect ratio
- **Secondary Logo (logo2.png)**: 160x40px or similar aspect ratio
